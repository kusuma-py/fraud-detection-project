# 🔍 AI-Based Cross-Chain Fraud Detection System

## 📌 Overview
This project focuses on detecting fraudulent transactions across blockchain networks using Artificial Intelligence and Machine Learning techniques. It analyzes transaction patterns and identifies suspicious activities using classification models and graph-based analysis.

---

## 🚀 Features
- Detects fraudulent and non-fraudulent transactions
- Uses Machine Learning algorithms for classification
- Performs data preprocessing and feature selection
- Visualizes transaction networks using graph analysis
- Provides performance metrics (Accuracy, Precision, Recall, F1-score)

---

## 🛠️ Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- NetworkX
- Matplotlib / Seaborn
- Jupyter Notebook
